# Work Partner · 电子宠物进化与任务系统

本仓库：[jmwasky-creative/work-partner](https://github.com/jmwasky-creative/work-partner)

## 项目简介

**Work Partner** 是一套**电子宠物进化游戏**与**任务发布/协作**相结合的系统：玩家养育并进化虚拟宠物，同时可以**创建、发布与完成任务**，将游戏成长与现实或社区目标绑定。

> 当前阶段：**仓库与任务框架已建立**；详细**设计文档**将在后续迭代中补充。

## 核心概念（草案）

| 方向 | 说明 |
|------|------|
| 电子宠物 | 可进化、成长、状态变化的虚拟伙伴 |
| 进化 | 通过玩法、时长、成就或任务完成度驱动形态/能力变化 |
| 任务 | 可由系统或用户发布，完成后影响宠物或玩家进度 |
| 协作 | 任务可面向个人或多人（具体规则待设计文档确定） |
